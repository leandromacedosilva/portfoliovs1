meus projetos

store dev

🍎 Apple Products Landing Page

Este projeto consiste no desenvolvimento de uma Landing Page moderna e responsiva inspirada no design minimalista utilizado pela Apple para apresentação de seus produtos. A página foi criada com o objetivo de demonstrar habilidades em desenvolvimento Front-End, organização de layout e construção de interfaces visuais atrativas.

O projeto foi desenvolvido utilizando tecnologias fundamentais do desenvolvimento web:

  • HTML5 para estruturação semântica da página
  • CSS3 para estilização, layout e responsividade
  • JavaScript para interatividade e comportamento dinâmico da interface
